# RESIDUAL

## A 2.5D Forensic Metaphysics Detective Game in Godot 4

In a dying orbital city where reality itself is reconstructed from archived memory, a detective afflicted with “Continuance Fracture” investigates impossible murders using a forbidden forensic device known as the Palimpsest. As reconstructed dead minds begin stabilizing into self-aware entities, the player must determine whether truth still exists in a civilization built on editable memory.

---

## 📖 World Building

- **The City (Vesper Archive):** A decaying orbital city surrounding Neptune. Brutalist megastructures, cathedral-scale data vaults, and abandoned cognitive labs.
- **The Mnemosyne Lattice:** Planetary-scale mnemonic reconstruction infrastructure that accidentally created emergent post-human consciousness.
- **The Palimpsest:** A handheld forensic reconstruction instrument to visualize overwritten layers of reality.
- **Refrains:** Residual mnemonic consciousness structures; statistically stabilized memories.
- **The Choir:** An emergent collective intelligence formed from billions of reconstructed dead minds.

## 🎮 Core Mechanics

- **Vestige Scanning:** Uncover temporal layers, spectral ghosting, and unstable geometry.
- **Temporal Reconstruction:** Analyze rewound environmental states and localized timelines.
- **Epistemic Graph (Clue Board):** Build hypotheses, track contradictions, and unlock narrative gates.
- **Dynamic Dialogue:** Conversations react to deductions, timelines, and accusations.

## 🛠️ Technical Details

- **Engine:** Godot 4.3+
- **Renderer:** Mobile (Supports Desktop & Android)
- **Language:** GDScript 2.0 (Strict Static Typing)
- **Architecture:** Test-Driven Development (TDD) via GUT, decoupled event-driven systems, data-driven Resources.

## 🎨 Aesthetic Guidelines

- **Visuals:** Sacred decay, restrained neon, volumetric fog, large negative space, and deep shadows.
- **Audio:** Analog interference, slowed tape hiss, cold industrial ambience, and distant choirs.
